# Data Types

There are two main data types, primitive and reference:

```
┌───────────────────────────────────────────────────┐
│                  Java Data Types                  │
│                         │                         │
│          ┌──────────────┴──────────────┐          │
│          │                             │          │
│    Primitive Types               Reference Types  │
│          │                             │          │
│  ┌───────┴───────┐             ┌───────┴───────┐  │
│  │ byte          │             │ Classes       │  │
│  │ short         │             │ Interfaces    │  │
│  │ int           │             │ Arrays        │  │
│  │ long          │             │ Enums         │  │
│  │ float         │             └───────────────┘  │
│  │ double        │                                │
│  │ boolean       │                                │
│  │ char          │                                │
│  └───────────────┘                                │
└───────────────────────────────────────────────────┘
```
